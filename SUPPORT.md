# Support

## Where to Ask Questions

For usage questions, integration help, and general discussion, use the project
issue templates or GitHub Discussions when available.

Use the following issue template for the right kind of request:

- bug reports: `.github/ISSUE_TEMPLATE/bug_report.yml`

For feature requests, improvements, and integration questions, open a plain
GitHub issue with a clear title and description.

## What to Include

Please include the package affected, Flutter and Dart versions, platform, and a
clear description of what happened.

If the problem involves logs or screenshots, redact any real secrets, tokens, or
passwords before posting them.

## A Quick Reminder

Never paste real secrets into public issues, discussions, or support requests.
If you suspect a security problem, use `SECURITY.md` instead of a public issue.
